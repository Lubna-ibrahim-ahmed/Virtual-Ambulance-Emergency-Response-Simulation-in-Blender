## 1. Project Overview

This project is an interactive 3D emergency-response simulation built in **Unity 6 using URP**.  
It presents four accident scenarios in a shared city environment, each connected to an ambulance rescue workflow.

The project demonstrates:

- 3D scene construction
- Lighting design
- Character animation
- Physics simulation
- Particle effects
- Audio integration
- C# event-driven gameplay logic
- UI interaction
- Version control using Git and Git LFS

The current project includes four emergency scenarios:

1. Gas Explosion
2. Car Collision
3. Stretcher Rescue
4. Falling Tree Accident

Each scenario is designed to show an accident event, witness reaction, emergency response, patient rescue, and ambulance evacuation.
